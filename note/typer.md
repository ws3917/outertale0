# 打字机格式定义：
- \` 控制符起点，\`控制符终点
- `c + 大写字母` 临时改变颜色，如`cY` `cB` `cR`，使用`c0`恢复
- 红橙黄绿青蓝紫黑白 = ROYGCBPKW
- 文本内直接`\n`换行，不是`\\n`
- `p + 数字` 停顿0.1s，比如`p1`停顿0.1s，`p2` 停顿0.2s
- `t + 数字` 临时改变速度，数字代表出字时间间隔，比如`t2`即0.5倍速，数字支持小数
- `t0` 代表后续内容即时显示，`t1`恢复正常速度
- 行为：如果
